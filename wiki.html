Start Setup
TMDB

Step 1: Create a TMDB key

Create a TMDB account

Go to the TMDB sign-up page, create an account, and sign in. A desktop browser is easiest for this part.

Open the API page in your TMDB settings

In TMDB, open your account settings and choose API. After signing in, this settings link may take you there directly.

Request a key

Agree to TMDB's terms and fill out the short form. If TMDB asks what kind of key you need, choose Developer for a personal app. That label is just TMDB's wording; you do not need to be a programmer.

Copy the right value

Look for API Read Access Token. Copy that long token and paste it into NightFlix where it asks for the TMDB Read Access Token.

Keep it private

Do not post your token online or share screenshots of it. If you think someone else has it, make a new one in TMDB and update NightFlix.

Quick tip

TMDB may also show an older value called an API Key. NightFlix wants the API Read Access Token, not the shorter API Key.

Movie Sources

Step 2: add a source

NightFlix does not include movies by itself. If you add a source, make sure it is safe, legal, and trusted.

Fixes

Common Problems

Nothing loads

Check that your TMDB Read Access Token is pasted correctly in Settings. It should be the long token from TMDB, not your TMDB password.

Play does not work

Add a safe movie source in Settings. If the source looks suspicious or redirects, remove it and use a trusted one.

Information looks old

Clear the saved TMDB information in Settings. NightFlix will download fresh information when needed.

You want to start over

Use the reset option in Settings to remove your saved TMDB token and movie source from this device.

Your setup stays on your device

NightFlix saves your token and movie source locally so you do not have to type them every time. Still, treat your TMDB token like private information.
